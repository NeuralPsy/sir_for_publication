
# Scale-Invariant Resonance (SIR) Framework Code
Jupyter notebook with simulations (Kuramoto, Ising, Hopfield) and EEG analysis from the paper "Scale-Invariant Resonance: A Multiscale Information-Theoretic Framework for Pattern Emergence" by Kirill Vasilchenko.

Raw EEG data: public UCI repository [[link](https://raw.githubusercontent.com/jbrown81/EEG-Eye-State-Dataset/master/EEG%20Eye%20State.csv)].
